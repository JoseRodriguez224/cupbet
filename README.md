# CupBet — FIFA World Cup 2026 Prediction Markets

Polymarket-style World Cup prediction markets with live scores and MetaMask USDT funding.

**Live site:** https://joserodriguez224.github.io/betting-site/

## Local development

```bash
npm install
npm run dev
```

## Deploy

Push to `main` branch — GitHub Actions builds and publishes to GitHub Pages automatically.

```bash
git push origin main
```
